The CppDepend source explorer permits to navigate across the files having the issues.
To open a file go to the issues tab and click on a file containing the issues.



This advanced explorer provides the following features:

Issues navigation: We can navigate easily across the file issues from the toolbar.
Issues evolution:We can explore the new/fixed/unresolved issues.
Labels: With labels we can get useful metrics about the methods.
Diff: We can explore the diff with the baseline.
Dependencies navigation: The dependencies of a method are clickable if they contain issues.
