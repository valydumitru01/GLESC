Detailed Description

Class representing a cubemap texture. It abstract away the creating and handling of cubemaps.

A cubemap is a type of texture that contains 6 faces of a cube, each face being a 2D texture.

A cubemap helps retrieving a texel from a 3D direction vector, which is useful for many rendering techniques

Example of cubemap usage is skyboxes, reflections, refractions, etc.
