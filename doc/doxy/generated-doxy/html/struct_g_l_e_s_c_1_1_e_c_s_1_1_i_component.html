Detailed Description

Base class for all components.

All components must inherit from this class and must implement all the pure virtual functions. This is necessary for the derived classes to be used in the ECS.

std::vector< EntityStatsManager::Value > & GLESC::ECS::IComponent::getDebuggingValues ( )
inline

Get the debugging values of the component.

This function is used to get the debugging values of the component. The values are stored in a vector of EntityStatsManager::Value. This vector is used by the EntityStatsManager to display the debugging values of all the components of an entity. This will be used by the HUD to display the values of each component and allow mid-game debugging and modifications. This function is called only once when the values vector so it will not be called every frame and there will be not update.

Returns
The debugging values of the component

virtual std::string GLESC::ECS::IComponent::toString ( ) const

Each component must have a toString function.

This is needed because when we want to debug the content of the components, we need to be able to do it without knowing the specific type of the component. This is because we are using a vector of IComponents.
