Detailed Description

This is the render engine of the game engine. It is resposable of sending the data to the GPU and render it.

The public methods are the update side of the renderer (with some exceptions). It must be executed in the update loop of the engine. While the private methods are the render side of the renderer. It must be executed in the render loop of the engine.
