Detailed Description

Set of macros for runtime and compile time assertions.

Author
Valentin Dumitru
Date
2023-09-26

This file contains a set of macros that can be used to check conditions at runtime and compile time. It's separated into two categories: runtime asserts and static asserts. If the NDEBUG macro is defined, the asserts will be empty, otherwise, they will throw an AssertFailedException if the condition is not met.

Compares two values for equality.

This function is used to compare two values for equality. It is only used for the runtime asserts. It already exists a similar implementation of equality comparison in Math.h, but given that we need asserts to be independent to avoid circular dependencies, we have to implement it here.
