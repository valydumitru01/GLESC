inlinestatic

Calculate the model matrix from the position, rotation, and scale.

This function calculates the model matrix from the position, rotation, and scale. The function applies the transformations in the following order: scale, rotation, and translation. The function assumes a specific order of transformations: scale, then rotation, then translation. As we're working with row-major matrices, the order of transformations in matrix multiplications is reversed.

Template Parameters
ModelTypeThe data type of the matrix and vector elements (e.g., float, double).
TypePosThe data type of the position vector elements (e.g., float, double).
TypeRotThe data type of the rotation vector elements (e.g., float, double).
TypeScaleThe data type of the scale vector elements (e.g., float, double).
Parameters
positionA 3D vector containing the position of the model.
rotationRadsA 3D vector containing the rotation angles (in radians) around X, Y, and Z axes respectively.
scaleA 3D vector containing the scale factors for the model.
modelA 4x4 matrix which will contain the result of the model matrix.
