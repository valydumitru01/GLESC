bool Line::intersects ( const Point &  pointParam) const

Check if a line intersects with a point.

This function checks if the point is on the line by checking if the cross product of the direction of the line and the vector from the point to the line is the zero vector. It uses an epsilon value to account for floating point errors.

Parameters
pointParamThe point to check for intersection.
Returns
true If the point is on the line.
