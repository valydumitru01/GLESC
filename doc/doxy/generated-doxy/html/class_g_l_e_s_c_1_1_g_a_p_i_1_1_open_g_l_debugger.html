static void GLESC::GAPI::OpenGLDebugger::setSdlGlDebugContextAttribute ( )
inlinestatic

Sets the SDL_GL_CONTEXT_DEBUG_FLAG attribute.

This attribute is used to enable OpenGL debug context, which is required for the OpenGL debug callback. This might reduce performance. It must be called before creating the OpenGL context.

See also
https://wiki.libsdl.org/SDL_GLattr#SDL_GLattr
