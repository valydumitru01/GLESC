template<typename Type, size_t N, size_t M>
class GLESC::Math::Matrix< Type, N, M >

Matrix class. Represents a matrix of N rows and M columns.

This matrix is row-major.

Template Parameters
TypeThe type of the elements in the matrix
NThe number of rows in the matrix
MThe number of columns in the matrix
