Enumeration Type Documentation

◆ RenderType

enum class GLESC::Render::RenderType
strong

This is the possible rendering types for a mesh.

Enumerator
SingleDrawStatic 

Single draw meshes are going to call render once per mesh every frame Static meshes are not going to change their vertices.

SingleDrawDynamic 

Single draw dynamic meshes can change their vertices at any point. Dynamic meshes can change their vertices at any point.

InstancedStatic 

Instanced meshes are going to be rendered multiple times with the same vertices Static meshes are not going to change their vertices.

InstancedDynamic 

Instanced dynamic meshes can change their vertices at any point. Dynamic meshes can change their vertices at any point.

BatchedStatic 

Batched meshes are going to be batched in a single large mesh and then rendered once (saving a lot of draw calls). Static meshes are not going to change their vertices.

BatchedDynamic 

Batched meshes are going to be batched in a single large mesh and then rendered once (saving a lot of draw calls). Dynamic meshes can change their vertices at any point.
