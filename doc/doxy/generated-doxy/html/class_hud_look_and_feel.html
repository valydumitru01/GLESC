Detailed Description

Class that encapsulates the look and feel of the HUD.

Allows setting the look and feel of the HUD, such as the font, font size, and colors.

Adds a font from a file with a size.

The font is added to the list of fonts that can be used in the HUD. Fonts need to be loaded before use, and this class handles the loading and caching of fonts.
