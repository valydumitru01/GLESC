Detailed Description

The InputManager class handles user input events such as keyboard and mouse events. It provides methods to initialize the input manager, update the input state, and check if a specific key is pressed. The class also handles event handling functions for keyboard and mouse events. The mouse position and key map are stored as private member variables.
