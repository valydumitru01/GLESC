Detailed Description

Color in RGBA values (x = [0] = R, y = [1] = G, z = [2] = B, w = [3] = A)

Will store colors in the range of 0 to 255

Has minimum overhead as only validates in debug mode.

Contains a set of static colors for convenience. Examples: ColorRgba::Red, ColorRgba::Green, etc.

Inherits from Vec4F to allow for easy conversion to Vec4F, as many functions expect Vec4F
