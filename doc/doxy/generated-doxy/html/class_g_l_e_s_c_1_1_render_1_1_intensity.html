Detailed Description

template<typename IntensityType>
class GLESC::Render::Intensity< IntensityType >

Intensity class, to encapsulate values between 0 and 1.

This ensures values are always between 0 and 1, as are very common in the engine. It has minimum overhead, and only validates in debug mode.

Template Parameters
IntensityTypeThe type of the intensity (float, double, etc)
