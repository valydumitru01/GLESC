Detailed Description

Color in RGB values.

Will store colors in the range of 0 to 255

Has minimum overhead as only validates in debug mode.

Contains a set of static colors for convenience. Examples: ColorRgb::Red, ColorRgb::Green, etc.

Inherits from Vec3F to allow for easy conversion to Vec3F, as many functions expect Vec3F
