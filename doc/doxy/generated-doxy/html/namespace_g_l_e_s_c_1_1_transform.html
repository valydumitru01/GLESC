Detailed Description

Namespace that contains all the classes and functions related to the Transform subsystem.

The Transform subsystem is responsible for managing the position, rotation, and scale of entities in the game world. Also stores functinality for transforming meshes and bounding volumes.

◆ HomogeneousPosition

using GLESC::Transform::HomogeneousPosition = typedef Vec4F

The homogenized position in 3D space, the fourth component is the w component.

The homogenized position in 3D space represented by a vector of 4 components The x, y, z and the w component The w component allows positions to be transformed by 4x4 matrices. Without the w component, we wouldn't be able to translate positions at the same time as rotating them and scaling them. To dehomogize positions we divide the x, y, z components by the w component.
