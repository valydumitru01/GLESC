void Skybox::draw ( const View &  view,
const Projection &  projection 
) const

Renders the skybox around the player given the view and projection.

A skybox is a cubemap that is always aroud the player and simulates infinity distance. This is done by having the skybox follow the camera and having the depth component in the shader to be maximum depth (1.0).
