◆ toString()

template<typename Type >
static std::string GLESC::Stringer::toString ( const Type &  value)
inlinestatic

Converts an object to a string.

The object to convert can be iterable or non iterable. If it is iterable, the output will be a list separated by commas. If it is non iterable, the output will be the string representation of the object. If the object is a floating point number, the precision of the output will be either the default or the one set by setFloatPrecision. Once the object is converted, the precision will be reset to the default. The default precision is 6.

Parameters
value- the object to convert.
Returns
the string representation of the object.
