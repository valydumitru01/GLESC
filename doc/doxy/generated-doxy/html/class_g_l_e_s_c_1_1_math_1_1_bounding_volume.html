Detailed Description

A bounding volume is a geometric shape that encloses a set of points.

This class updates the bounding cuboid to fit the points it encloses.

struct GLESC::Math::BoundingVolume::AABB

The axis-aligned bounding box of the bounding volume.

AABB is a box aligned with the axes of the coordinate space. It is a very simple shape that can be used to enclose other shapes. It's simpicity makes operations like intersection and containment very fast. [https://gdbooks.gitbooks.io/3dcollisions/content/Chapter1/aabb.html]
