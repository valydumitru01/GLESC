constexpr Scalar GLESC::Physics::PhysicsManager::velocityScalar {0.001f}
staticconstexprprivate

This scalar is used to scale the velocity of the object.

We need this because using fixed update time steps makes normal values too bit, and the object would move too fast. Using a scalar makes the used be able to handle more realistic numbers.
