struct GLESC::Math::SelectSmallerFloatingType
template<typename T1, typename T2>
struct GLESC::Math::SelectSmallerFloatingType< T1, T2 >

Selects the smaller floating-point type between two types.

The function selects the smaller floating-point type between two types. If both types are floating-point, it selects the type with the smaller size. If only one type is floating-point, it selects that type.

Template Parameters
T1The first type
T2The second type
Returns
The smaller floating-point type between the two types
