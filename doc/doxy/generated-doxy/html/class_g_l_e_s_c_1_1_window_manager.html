void WindowManager::setFullscreenBorderless ( bool  isFullScreen)

Sets the window fullscreen borderless or "fake fullscreen".

This will make the window fullscreen but also facilitates changing to different apps and windows. This won't change the resolution of the screen, like true fullscreen. But also is less efficient than true fullscreen, costing more resources.

isFullScreentrue to enable, false to disable
